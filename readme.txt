== Description ==

Management system for small businesses and self-employed.

Manage your customers, budgets and invoices from the administration of your web Wordpress.

Get the most out of your website by creating your own cloud management. Access your data from anywhere, saving time and money.

You can manager:
> * Clients
> * Budgets
> * Invoices

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/plugin-name' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Gestion-Pymes screen to configure the plugin

== Frequently Asked Questions ==

= Can I generate the invoices/budgets pdf ? =

Yes, you can do it from the invoice/budget edit page.
